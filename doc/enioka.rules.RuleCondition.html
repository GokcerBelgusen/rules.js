Interface: RuleCondition

enioka.rules.RuleCondition



A Rule Condition is a Rule Element that hold conditions of a rule. A number of predefined conditions are built in in the core rules implementation:
- enioka.rules.conditions.NOT
- enioka.rules.conditions.OR
- enioka.rules.conditions.AND
- enioka.rules.conditions.ISNULL
- enioka.rules.conditions.LESS
- enioka.rules.conditions.MORE
- enioka.rules.conditions.MATCHES
- enioka.rules.conditions.EQUALS
- enioka.rules.conditions.LIKE
One may extend these predefined conditions by providing additional condition handlers at engine initialization. It is as simple as providing either a named set of classs with the specified RuleCondition interface, with the handler interface or simply of functions with the proper interface.
