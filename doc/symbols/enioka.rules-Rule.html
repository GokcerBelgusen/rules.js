<inner> enioka.rules-Rule(ruleXML, father, id)
Parameters:
ruleXML
The source code for the rule as an XML object
father
The embedding father rule if any (as an object)
id
The unique id allocated to identify this rule (integer counter). This this takes an XML source code rule and registers "fast" access to the conditions actions and subrules. It does propagate the rule creation to embedded rules as well. See Rule syntax in documentation for further information on rule syntax.
