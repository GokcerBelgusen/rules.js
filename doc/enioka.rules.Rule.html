Class: Rule

enioka.rules.Rule

This class represents the rules objects of the engine. These objects are directly mapped against their XML source, without (for the moment) compilation to an alternative form more efficient or convenient for execution.

This class is not exposed either to the client API. It is only "seen" from the client application as an XML object that customizes the behaviour of the engine in a "declarative" form.
